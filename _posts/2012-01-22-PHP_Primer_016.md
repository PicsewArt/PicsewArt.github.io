在 PHP 中支持三种注释风格:

* `//` : 双斜杠单行注释，注释内容写在后面，不支持换行

* `#` : 井号单行注释，注释内容写在后面，不支持换行

* `/*` 与 `*/` : 多行注释，注释内容中支持换行

举个栗子:

```php

<?php 
	$var = "normal code"; // normal code here
	// example:
	$a = 0;
	$b = 1; # another example
	$c = 2;
	/*
	if ($a == 0) {
		echo "a=0";
	}
	*/
	$d = 3;
	$e = 4;
?>

```

如果在 `<?php` 和 `?>` 之外的 HTML 代码中添加注释，应当使用 HTML 的注释语法，即 `<COMMENT>` 与 `</COMMENT>` 组合或 `<!--` 和 `-->` 组合。
