资源(即 `resource`)由专门的函数来建立和使用，如打开文件、数据连接等。举个栗子:

```php

<?php
$file = fopen("test.txt","r");   // 打开文件
$con = mysql_connect("localhost","root","root");  // 连接数据库
$img = imagecreate(200,200);// 图形画布
?>

```
