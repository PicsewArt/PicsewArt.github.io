### 获取 Web 页面内容高度

利用 JavaScript 来实现，一般写在 UIWebView 代理中。

```objc

- (void)webViewDidFinishLoad:(UIWebView *)webView  {
    CGFloat height = [[webView stringByEvaluatingJavaScriptFromString:@"document.body.offsetHeight"] doubleValue];
		// ...
}

```
