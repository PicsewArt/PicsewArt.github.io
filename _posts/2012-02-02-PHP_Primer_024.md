上一篇学会了线条的绘制，这一篇来了解一下如何绘制文字。

## 画布与颜色

依然要创建画布和颜色:

```php

<?php
$img=imagecreatetruecolor(100, 100);
$red = imagecolorallocate($img, 0xFF, 0x00, 0x00);

```


## 绘制文字

```php

imagestring($img, 5, 0, 0, "Hello, world", $red);

```


## 输出图片释放内存

```php

header("content-type: image/png");
imagepng($img);
imagedestroy($img);
?>

```
