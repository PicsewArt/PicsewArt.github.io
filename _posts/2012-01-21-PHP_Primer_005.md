整型(即 `integer`)可以用十进制、八进制、十六进制指定。其中八进制数字前必须加上阿拉伯数字零`0`；十六进制数字前必须加 `0x`:

```php

<?php
$data_int = 1234;   // 十进制数 正数
$data_int = -123;    // 十进制数 负数
$data_int = 0123;    // 八进制数 (等于十进制的 83) 
$data_int = 0x1a;    // 十六进制数 (等于十进制的 26) 
?>

```
